# Power BI Dashboard Report

## Project Overview
This project contains a Power BI dashboard designed to analyze and visualize business data through interactive charts, KPIs, and filters. The report provides insights using multiple visualizations such as line charts, bar charts, donut charts, cards, and slicers.

The dashboard was created using **Microsoft Power BI** and is stored in the file:

`First Report.pbix`

---

# Features

- Interactive dashboard for data analysis
- KPI cards for quick business insights
- Line charts for trend analysis
- Bar charts for category comparison
- Donut charts for percentage distribution
- Slicer for dynamic filtering
- User-friendly report layout

---

# Dashboard Components

The report includes the following visual elements:

| Visual Type | Purpose |
|---|---|
| Line Chart | Analyze trends over time |
| Bar Chart | Compare categories and values |
| Donut Chart | Show proportional distribution |
| KPI Cards | Display key metrics |
| Slicer | Filter dashboard data interactively |
| Combo Chart | Compare multiple measures together |

# How to Open the Project

1. Install Microsoft Power BI Desktop.
2. Download or clone this project folder.
3. Open the file `First Report.pbix` using Power BI Desktop.
4. Refresh the dataset if required.
5. Explore the dashboard using filters and slicers.
